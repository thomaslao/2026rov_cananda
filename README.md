# ROV Task Manager

This repository uses **v16** as the primary ROV Task Manager app.

## Open The App

- Default entry: `index.html`
- Direct v16 app: `v16/ROV_Task_Manager_v16.html`
- Local launcher: `OPEN_V16.html`

v16 is a standalone local-first app. It can be opened directly in a browser and keeps normal work in browser localStorage unless a guarded Supabase sync is manually confirmed.

## Version Notes

- `v16/` is the primary working version.
- `ROV_Task_Manager_v15.html` is kept as the previous fallback version.
- v16 release details are in `v16/README.md` and `v16/RELEASE_NOTES.md`.
