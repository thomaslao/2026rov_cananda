Task 1.2 Coral Garden Modeling - Photogrammetry Technical Stack
最高得分：40分 | 场地：Flume Tank（有水流）

📊 评分方法对比

方法 最高分 核心要求
摄影测量 Photogrammetry（自动） 40分 彩色目标方块可见、长度已量测验证、比例正确、高度误差 ≤5 cm
手工 CAD 建模 30分 长度与高度误差各 ≤5 cm，可旋转 3D 模型
纸面三视图 20分 长度与高度误差各 ≤5 cm，正、俯、侧三视图
️ 重要：三种方法可同时进行，但只计最高分一种。

🔧 推荐技术栈（Photogrammetry 路线）

📷 硬件层

  • 1080p@30fps 广角镜头（FoV ≥ 90°）
  • 防水 ArUco marker（≥4×4 cm，≥8个）
  • 2×LED 水下灯（≥1000 lm）
  • 6 推进器配置（5自由度以上）
  • MS5837-30BA 深度传感器

💻 软件层 - 影像采集

  • GStreamer / OpenCV VideoCapture
  • 视频录制：H.264 MP4，≥30fps，1080p
  • libcamera / Pi Camera 驱动
  • UDP 串流 → 5600 port
  • 时间戳记同步（NTP 或 PPS）

🎨 软件层 - 3D 建模

  • Meshroom（首选，GPU加速）
  • COLMAP（备用，CLI友好）
  • OpenDroneMap（Docker部署）
  • CloudCompare（必装，量测用）
  • Python 3.10+（opencv-python, numpy）

️ 软件层 - CAD 备案

  • FreeCAD（免费，跨平台）
  • Fusion 360（教育版）
  • Tinkercad（最快上手）
  • 目标：5分钟内完成基本模型

🎯 目标方块规格（Photogrammetry 必要）

类型 ArUco DICT_4X4_50 或彩色编码圆形标靶
尺寸 5×5 cm（实体），防水护贝或防水打印
数量 ≥8 个（分布于前、中、后、左、右、顶部）
固定方式 磁吸底座 / 吸盘，赛前由操作员摆放
已知距离 两个标靶间距需事先量测记录（作为比例尺）

操作 SOP（15 分钟内完成）

赛前准备（入水前）

  • 确认目标板已摆放，记录两板间已知距离
  • 确认摄影机白平衡锁定（避免曝光变化）
  • 开始录制视频（1080p@30fps）

扫描飞行（入水后 ~3 分钟）

  • Z 字形扫描：从左→右，前→后，覆盖所有目标板
  • 补充环绕扫描：围绕珊瑚礁顶部一圈（确保覆盖高度）
  • 确认每个目标板至少出现 3 次以上画面

离水后建模（~8 分钟）

  • ffmpeg 拆幀：每秒取 5 幀
  • 模糊侦测：删除低质量幀
  • Meshroom 建模：GPU 模式 ~5 分钟
  • CloudCompare 量测：测量 L 和 H，验证误差 ≤5 cm

赛场交付

  • 旋转展示 3D 模型给评分官
  • 报告 L（长度）和 H（高度）量测值
